# 💼 Job Application Tracker

A web application built using Django to manage and track job applications efficiently.

## ✨ Features
- Add, Edit, View, and Delete Job Applications
- Live status tracking (Applied, Interview, Offer, Rejected)
- Responsive UI built with Bootstrap 5
- Clean Django MVT Architecture

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Frontend:** HTML, CSS, JavaScript, Bootstrap 5
- **Database:** SQLite
